# **App Name**: Mi Amor Eterno

## Core Features:

- Principal pregunta: Mostrar la pregunta '¿Quieres ser mi San Valentín?' centrada en la pantalla.
- Botón de 'Sí': Un botón verde que al hacer clic lleve a una página de confirmación.
- Botón de 'No' dinámico: Un botón rojo que, al hacer clic, inicie una animación donde el botón verde de 'Sí' se agranda gradualmente con cada clic hasta cubrir toda la pantalla.
- Animación de crecimiento: Implementar una animación suave y fluida para el crecimiento del botón 'Sí'.
- Página de confirmación: Una página especial que muestre un mensaje de confirmación.

## Style Guidelines:

- Color primario: Rosa (#FF69B4), evocando el amor y la dulzura.
- Color de fondo: Blanco (#FFFFFF), que aporta pureza y sencillez.
- Color de acento: Verde (#32CD32) para el botón de 'Sí', representando esperanza y crecimiento.
- Fuente para el cuerpo del texto: 'Alegreya', fuente serif, para una sensación elegante e intelectual.
- Fuente para titulares: 'Belleza', fuente sans-serif, para darle personalidad a los encabezados
- Iconos: Corazones y flechas sutiles para complementar la temática romántica.
- Diseño centrado para enfocar la atención en la pregunta y las opciones.
- Animaciones suaves y fluidas en las transiciones y el crecimiento del botón.
